# Issue Analysis

This module enables issue-specific analysis of United Nations General Assembly voting behavior.

Researchers can analyze any issue or topic represented in UNGA resolutions.

Examples include:

- Peace and Security
- Human Rights
- Climate Change
- Disarmament
- Decolonization
- Sustainable Development
- International Law
- Nuclear Weapons
- Refugees
- Artificial Intelligence
- Global Health
- Territorial Disputes
- Armed Conflicts

Features:

- Resolution filtering
- Agreement matrices
- Community detection
- Centrality analysis
- Network statistics
- Publication-quality figures
- Temporal analysis

Example:

issue_analysis(
    issue="Any UNGA topic",
    start_session=1,
    end_session=79
)
